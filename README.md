Placement Data Analysis Project
Dataset Selected

Campus Placement Dataset

The dataset contains information about students' academic performance, work experience, specialization, MBA percentage, placement status, and salary packages.

Why I Selected This Dataset

I selected the Placement Dataset because it provides real-world business and educational insights. The dataset allows analysis of factors that influence student placements, salary outcomes, and specialization performance. It is suitable for applying data cleaning, SQL analysis, and dashboard visualization techniques.

Three Business Questions Answered
1. What is the overall placement rate?

Using Power BI and SQL analysis, I calculated the percentage of students who were successfully placed compared to the total number of students.

2. Which specialization has the highest average salary?

I analyzed salary trends across different specializations to identify which specialization generates the highest average salary package.

3. Does MBA percentage influence placement success?

I compared MBA percentages of placed and non-placed students using visualizations and summary statistics to determine whether academic performance impacts placement outcomes.

One Key Insight

Students with higher MBA percentages generally showed a higher placement success rate, and certain specializations achieved significantly higher average salary packages than others. This suggests that both academic performance and specialization choice can influence placement outcomes.
